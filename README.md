# Example
<h2>Draw Tangent Circles in Python Turtle</2>
<br>

```ruby
import turtle
```

```ruby
t = turtle.Turtle()
for i in range(50):
  t.circle(10*i)
  t.up()
  t.sety((10*i)*(-1))
  t.down()
```
